VISUALIZATION

This section includes charts and maps relating to our project, which might help to uncover some interesting insights about the dataset.

NUMBER OF REQUEST PER DATE

The bar chart demonstrates the number of requests for each day, from 3rd December 2018 to 9th December 2018.

The figures for 3rd December was the highest, which was fourfold the number of requests of the second place, 6th December.

NUMBER OF REQUEST PER FILE TYPE

The bar chart presents the number of requests for each file type from 3rd December 2018 to 9th December 2018.

There were only two different types of files, ts and m3u8, with m3u8 accounted for the majority of the requests.

PERCENTAGE OF REQUESTS FOR EACH PROVINCE

The pie chart presents the number of requests for each province (both in Vietnam and overseas) from 3rd December 2018 to 9th December 2018.

A large proportion of requests belongs to Northern provinces in Vietnam with Hanoi accounted for approximately two fifths of the total. Most of the overseas requests came from the United States and Germany.

PERCENTAGE OF REQUESTS FOR EACH ISP

The pie chart presents the number of requests for each Internet Service Provider (ISP) from 3rd December 2018 to 9th December 2018.

VNPT Corp had the highest figure with approximately fifty percents, followed by FPT Telecom Company. On the contrast, Cox Communication Inc. only accounted for one percent, which was the lowest one among the recored ISPs.
